## HELP

isekai
```
Commands:

    help [command...]         Provides help for a given command.
    exit                      Exits application.
    characters                Show available [AVATAR] saves.
    create [template] [name]  Create a new isekai project from [template] or @isekai/template
    logs [AVATARS...]         follow the active [AVATAR] logs
    run [...AVATARs]          run and watch [AVATAR] files
    status                    [STATUS] of active [AVATAR] files.
    stop [AVATARS...]         stop active [AVATAR] files.
```
